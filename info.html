💡 Recomendaciones para Obtener la Máxima Puntuación
Para Veracidad (5 pts): Lea cuidadosamente el caso y analice jurídicamente antes de responder.

Para Argumentación (5 pts):
  • Proporcione justificaciones detalladas (mínimo 50 palabras)
  • Use terminología jurídica apropiada
  • Cite principios o normativas relevantes cuando sea posible
  • Demuestre análisis crítico y razonamiento propio
  • Mantenga estructura lógica y coherente
  • Evite respuestas genéricas como "sí", "no", "tal vez"
Para Evitar Penalizaciones: No use herramientas de IA, evite copiar y pegar, escriba respuestas sustanciales.
